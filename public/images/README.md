# 🎨 Carpeta para colocar tus Logos y Sello Personal

¡Hola! Esta es la carpeta física donde puedes **arrastrar y soltar** tus propios logotipos e imágenes. 

He programado el catálogo con una tecnología inteligente de **auto-recuperación (Fallback)**. Esto significa que si colocas una imagen aquí con el nombre adecuado, la web la mostrará de inmediato. Si no colocas nada, el sitio seguirá mostrando los hermosos logotipos vectorizados SVG predeterminados sin romperse ni mostrar errores.

## 📌 Nombres de archivos que debes usar:

### 1. Tu Sello Personal / Marca principal
*   Guarda tu logotipo principal como: **`sello.png`** (también puedes usar `.jpg`).
*   **Ubicación:** `public/images/sello.png`
*   *Nota: Si este archivo no existe, la cabecera cargará automáticamente `/logo.png`.*

### 2. Logos de Canales y Aplicaciones Digitales
Coloca los logos de tus plataformas con los siguientes nombres exactos en minúsculas y formato `.png`:

*   🎬 **Netflix:** `netflix.png`
*   🏰 **Disney+ Premium:** `disney.png`
*   🏰 **Disney+ Estándar:** `disney_std.png`
*   🎥 **HBO Max (Max):** `hbo.png`
*   🛍️ **Amazon Prime Video:** `prime.png`
*   ⭐️ **Paramount+:** `paramount.png`
*   🎌 **Crunchyroll:** `crunchyroll.png`
*   🎵 **Spotify:** `spotify.png`
*   🤖 **ChatGPT:** `chatgpt.png`
*   🎨 **Canva:** `canva.png`

---

## ⚡ Consejos de Formato
- **Transparencia:** Te recomiendo usar imágenes con fondo transparente (formato **PNG**) para que se fusionen de forma espectacular con el fondo oscuro y los neones del catálogo.
- **Tamaño:** `512x512 píxeles` es el tamaño ideal para los logos de plataformas, y `1024x1024 píxeles` para tu sello.
